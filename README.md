# Aswin Das — Digital Marketer & Growth Strategist

A premium, performance-oriented digital marketing portfolio website built with a modern, high-contrast tech-minimalist design.

## Tech Stack
- **Core**: React 19, TypeScript
- **Bundler & Build Tool**: Vite 8
- **Styling**: Tailwind CSS v4 (monochromatic theme)
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Development

To start the local development server:
```bash
npm install
npm run dev
```

To build the static assets for production:
```bash
npm run build
```

## Deployment

This repository is configured for automatic deployment to **GitHub Pages** using GitHub Actions via the workflow file in `.github/workflows/deploy.yml` on push to the `main` branch.
